## Автоматизация

### Обновление индекса
```bash
# Автоматическое обновление индекса
/doc-index [название_папки]

# Проверка актуальности индекса
/doc-index [название_папки] --check
```

### Проверка документации
```bash
# Проверка здоровья документации
make docs-health

# Проверка только этой категории
make docs-check-[категория]
```
