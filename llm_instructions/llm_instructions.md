# Инструкции для LLM

В данной папке собраны инструкции для LLM, применимые для всего проекта.

## Список инструкций

| Файл | Описание |
|------|----------|
| `instructions_general_docs.md` | Правила ведения документации: структура general_docs/, workflow статусов, правила feedback |
| `instructions_scripts.md` | Описание служебных скриптов для поддержания порядка в проекте |

## Шаблоны документов

Папка `templates/` содержит рекомендуемые шаблоны для создания документов:

| Шаблон | Назначение |
|--------|------------|
| `template_discuss.md` | Шаблон дискуссии (discuss/) |
| `template_architecture.md` | Шаблон архитектуры (architecture/) |
| `template_imp_plan.md` | Шаблон плана реализации (imp_plans/) |
| `template_resource.md` | Шаблон ресурса (resources/) |
| `template_folder_doc.md` | Шаблон документации папки (*_doc.md) |

**Примечание:** Шаблоны — рекомендуемая структура. Адаптируйте под конкретную задачу.
