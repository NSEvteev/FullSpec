# [AMY-00003] Добавить линковку между диаграммами и архитектурными документами

**Статус:** pending

## Описание

Диаграммы в папке `general_docs/03_diagrams/` изолированы от архитектурных документов. Необходимо добавить ссылки на диаграммы в архитектурные документы и обратные ссылки из диаграмм на архитектуру.

## Контекст

**Источник:** Отчёт о глубоком смысловом аудите документации (llm_tasks/temp/amy-santiago/doc_health_report_2026-01-15.md)

**Проблема:** Визуальное представление (диаграммы) не связано с текстовым описанием (архитектура), что снижает ценность документации.

## Подзадачи

- [ ] Прочитать существующие диаграммы (000_diagrams.md, README.md)
- [ ] Определить, какие архитектурные документы должны ссылаться на диаграммы
- [ ] Добавить секцию "Диаграммы" в архитектурные документы (если есть)
- [ ] Добавить ссылки на диаграммы в формате `[Название диаграммы](../03_diagrams/папка/файл.drawio)`
- [ ] Добавить обратные ссылки из диаграмм (в README.md или комментариях)
- [ ] Обновить 000_architecture.md — добавить столбец "Диаграммы"
- [ ] Проверить корректность всех ссылок
