# Текущие задачи

## Назначение

Файл для отслеживания задач текущей сессии работы с LLM. Содержит активные и выполненные задачи в рамках некоторой рабочей сессии и/или обработки некоторой "фазы" разработки.

**Связанный файл:** [future_tasks.md](future_tasks.md) — задачи на будущее

## Статусы задач

| Статус | Описание |
|--------|----------|
| `[ ]` | Запланирована |
| `[~]` | В работе |
| `[x]` | Выполнена |
| `[!]` | Заблокирована |

## Формат записи

```markdown
- [статус] **Краткое название** — Описание задачи
  - Подзадача 1
  - Подзадача 2
  - Ссылка: [документ](путь/к/документу.md)
```

## Правила работы

1. **Начало сессии:** Просмотреть текущие задачи, взять в работу
2. **В процессе:** Обновлять статусы по мере выполнения
3. **Завершение:** Перенести выполненные в архив, новые идеи — в [future_tasks.md](future_tasks.md)
