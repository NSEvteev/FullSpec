## Правила

### Что проверяется

| Проверка | Описание |
|----------|----------|
| Существование цели | Файл/папка по ссылке существует |
| Корректность якоря | Заголовок с таким якорем есть в документе |
| Формат пути | Соответствует правилам из [format.md](./format.md) |
| Доступность | Внешние URL отвечают (опционально) |

### Что НЕ проверяется

- Внешние ссылки (по умолчанию)
- Ссылки в code blocks
- Ссылки в комментариях

## Типы проверок

### Проверка существования

```
Ссылка: [README](/src/README.md)
    │
    ├── Файл /src/README.md существует?
    │   ├── ДА → ✅ Валидно
    │   └── НЕТ → ❌ Битая ссылка
```

### Проверка якорей

```
Ссылка: [Шаги](./workflow.md#шаги)
    │
    ├── 1. Файл ./workflow.md существует?
    │   └── НЕТ → ❌ Битая ссылка
    │
    └── 2. В файле есть ## Шаги?
        ├── ДА → ✅ Валидно
        └── НЕТ → ⚠️ Битый якорь
```

### Проверка формата

| Правило | Валидно | Невалидно |
|---------|---------|-----------|
| Markdown расширение | `file.md` | `file` |
| Trailing slash для папок | `folder/` | `folder` |
| Нет пробелов в пути | `my-file.md` | `my file.md` |

## Битые ссылки

### Типы

| Тип | Описание | Пример |
|-----|----------|--------|
| Несуществующий файл | Цель удалена/перемещена | `[old](/deleted.md)` |
| Битый якорь | Заголовок изменён | `[x](#old-heading)` |
| Неверный путь | Опечатка в пути | `[x](/scr/file.md)` |

### Формат пометки

При обнаружении битой ссылки:

```markdown
<!-- До -->
[Удалённый файл](/old/path.md)

<!-- После (пометка /links-delete) -->
[Удалённый файл](/old/path.md) <!-- BROKEN: файл удалён -->
```

### Действия при обнаружении

| Ситуация | Действие |
|----------|----------|
| Файл перемещён | Обновить путь |
| Файл удалён | Удалить ссылку или заменить |
| Якорь изменён | Обновить якорь |

## Автоматизация

### Команда проверки

```bash
# Проверить все ссылки в проекте
/links-validate

# Проверить конкретный файл
/links-validate src/README.md

# Проверить с автоисправлением
/links-validate --fix
```

### Вывод

```
Проверка ссылок...

❌ /src/README.md:15
   [старый API](/docs/api-v1.md)
   → файл не существует

⚠️ /specs/README.md:42
   [Шаги](#шаги-воркфлоу)
   → якорь не найден (есть: #шаги, #воркфлоу)

✅ Проверено: 156 ссылок
❌ Ошибок: 1
⚠️ Предупреждений: 1
```
