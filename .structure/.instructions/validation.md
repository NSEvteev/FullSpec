## Правила

### Согласованность ссылок

Все ссылки в /.structure/README.md должны быть валидны:

| Проверка | Правило |
|----------|---------|
| Ссылки на README | Файл существует |
| Относительные пути | Формат `../папка/README.md` |
| Якоря в Оглавлении | Соответствуют заголовкам |

### Согласованность дерева

Дерево папок в секции "3. Дерево папок" должно соответствовать:

1. **Файловой системе** — все папки существуют
2. **Секциям** — все папки из секции "Корневые папки" в дереве
3. **Оглавлению** — все корневые папки в Оглавлении

### Согласованность описаний

Описания папок должны соответствовать [format.md](./format.md):

- Заголовок с иконкой 🔗
- Ссылка на README
- Краткое назначение жирным
- Описание содержимого

## Типы проверок

### Проверка ссылок

```
Для каждой ссылки [текст](путь):
1. Путь существует?
2. Если якорь (#section) — секция существует?
3. Если README — файл существует?
```

### Проверка дерева

```
Для каждой записи в дереве:
1. Папка существует в файловой системе?
2. Комментарий соответствует описанию в секции?

Для каждой папки в файловой системе:
1. Папка есть в дереве?
2. Если нет — добавить или это исключение?
```

### Проверка синхронизации

```
Оглавление ↔ Секции ↔ Дерево

1. Все папки из Оглавления есть в секциях?
2. Все папки из секций есть в дереве?
3. Порядок алфавитный?
```

## Примеры

### Валидная структура

```markdown
## Оглавление
- [.claude/](#-claude)        ✅ Есть секция
- [src/](#-src)               ✅ Есть секция

### 🔗 [.claude/](../.claude/README.md)    ✅ Файл существует
### 🔗 [src/](../src/README.md)            ✅ Файл существует

## 3. Дерево папок
├── .claude/                  ✅ Соответствует
└── src/                      ✅ Соответствует
```

### Ошибки валидации

| Ошибка | Причина | Решение |
|--------|---------|---------|
| Битая ссылка | README не существует | Создать README или убрать ссылку |
| Папка не в дереве | Забыли добавить | Добавить в дерево |
| Нет в Оглавлении | Забыли обновить | Добавить ссылку |
| Папка удалена | Не обновили SSOT | Удалить из всех мест |
