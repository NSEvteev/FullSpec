# Дискуссии

Папка для хранения дискуссий, заметок и анализов.

## Назначение

- Обсуждения архитектурных решений
- Анализы и исследования
- Заметки по задачам
- Временные документы

## Структура

```
/.claude/discussions/
├── README.md                           # Этот файл
├── {дата}-{тема}.md                    # Дискуссия
└── analysis-{дата}.md                  # Анализ
```

## Формат именования

```
{YYYY-MM-DD}-{краткое-описание}.md
```

Примеры:
- `2026-01-20-refactoring-skills.md`
- `analysis-2026-01-20.md`
