# Тесты: skill-migrate

## Smoke-тесты

### smoke-1: Справка

**Команда:** `/skill-migrate --help`

**Ожидание:** Вывод справки по использованию

**Проверка:**
- [ ] Показаны параметры
- [ ] Показаны примеры

---

### smoke-2: Dry-run существующего скилла

**Команда:** `/skill-migrate input-validate test-rename --dry-run`

**Ожидание:** План миграции без изменений

**Проверка:**
- [ ] Показан список файлов для изменения
- [ ] Вывод содержит ℹ️
- [ ] Файлы не изменены

---

## Функциональные тесты

### func-1: Полная миграция

**Подготовка:**
1. Создать тестовый скилл через /skill-create test-migrate-source

**Команда:** `/skill-migrate test-migrate-source test-migrate-target`

**Ожидание:** Скилл переименован

**Проверка:**
- [ ] Папка test-migrate-target существует
- [ ] Папка test-migrate-source не существует
- [ ] SKILL.md содержит новое имя
- [ ] skills.md обновлён

**Очистка:** `/skill-delete test-migrate-target`

---

### func-2: Изменение категории

**Подготовка:**
1. Создать тестовый скилл в категории utility

**Команда:** `/skill-migrate test-skill --category documentation`

**Ожидание:** Категория изменена

**Проверка:**
- [ ] SKILL.md содержит `category: documentation`
- [ ] skills.md обновлён с новой группировкой

---

### func-3: Обновление ссылок

**Подготовка:**
1. Создать скилл A
2. Создать скилл B со ссылкой на A

**Команда:** `/skill-migrate A A-renamed`

**Ожидание:** Ссылки в B обновлены

**Проверка:**
- [ ] B/SKILL.md содержит ссылку на A-renamed
- [ ] Нет ссылок на старое имя A

---

## Edge cases

### edge-1: Скилл не существует

**Команда:** `/skill-migrate not-exists new-name`

**Ожидание:** Ошибка

**Проверка:**
- [ ] Вывод содержит ❌
- [ ] Причина: "Скилл not-exists не найден"

---

### edge-2: Новое имя занято

**Команда:** `/skill-migrate skill-create skill-update`

**Ожидание:** Ошибка

**Проверка:**
- [ ] Вывод содержит ❌
- [ ] Причина: "Скилл skill-update уже существует"

---

### edge-3: Миграция критичного скилла

**Команда:** `/skill-migrate skill-create skill-create-v2 --dry-run`

**Ожидание:** Предупреждение о критичности

**Проверка:**
- [ ] Вывод содержит ⚠️
- [ ] Текст: "Критичный скилл"
- [ ] Требуется подтверждение
