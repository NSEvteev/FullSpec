# Структура проекта

Команда для генерации и обновления структуры проекта в документации.

## Файлы для обновления

| Файл | Раздел |
|------|--------|
| `llm_instructions/llm_instructions.md` | `## Структура проекта` |
| `README.md` | `## Структура проекта` |

## Инструкции

### Шаг 1: Собрать структуру проекта

Используй Glob для получения списка файлов и папок проекта.

**Исключить из вывода:**
- `.git/` — служебная папка git
- `node_modules/` — зависимости npm
- `__pycache__/` — кэш Python
- `.venv/`, `venv/` — виртуальные окружения
- `*.pyc`, `*.pyo` — скомпилированные файлы Python
- `.DS_Store` — системные файлы macOS

**Включить:**
- Все `.md` файлы
- Все `.py`, `.js`, `.ts` файлы
- Все `.json`, `.yaml`, `.yml` конфигурации
- Папки со значимой структурой (с комментариями)
- `.gitkeep` файлы (показывают пустые папки)

### Шаг 2: Сформировать древовидную структуру

Формат вывода:
```
trading_platform/
├── файл.md                    # Краткое описание
├── папка/                     # Описание папки
│   ├── подпапка/              # Описание
│   │   └── файл.md
│   └── файл.py
└── последний_элемент/
```

**Правила форматирования:**
- Использовать символы: `├──`, `└──`, `│`
- Папки заканчиваются на `/`
- Комментарии через `#` после имени
- Сначала папки, потом файлы (в алфавитном порядке)
- Максимальная глубина: 4 уровня

### Шаг 3: Обновить llm_instructions.md

Найти раздел `## Структура проекта` в файле `llm_instructions/llm_instructions.md`.

Заменить блок кода (``` ... ```) внутри этого раздела.

### Шаг 4: Обновить README.md

Найти раздел `## Структура проекта` в файле `README.md`.

Заменить блок кода (``` ... ```) внутри этого раздела.

### Шаг 5: Вывести результат

После обновления файлов вывести:
1. Обновлённую структуру проекта
2. Сообщение об успешном обновлении обоих файлов

## Пример вывода

```
trading_platform/
├── .claude/                       # Конфигурация Claude Code
│   ├── agents/                    # AI-агенты
│   └── skills/                    # Скиллы
├── general_docs/                  # Общая документация
│   ├── glossary.md                # Глоссарий терминов
│   └── ...
├── llm_instructions/              # Инструкции для LLM
│   └── ...
├── CLAUDE.md                      # Быстрый справочник Claude
└── README.md                      # Описание проекта

✓ Структура обновлена в llm_instructions/llm_instructions.md
✓ Структура обновлена в README.md
```
