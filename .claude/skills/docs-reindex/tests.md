# Тесты: docs-reindex

## Smoke-тесты

### smoke-1: Проверка без изменений

**Команда:** `/docs-reindex --check`

**Ожидание:** Отчёт о состоянии

**Проверка:**
- [ ] Показана статистика ссылок
- [ ] Показано состояние README
- [ ] Файлы не изменены

---

### smoke-2: Dry-run

**Команда:** `/docs-reindex --dry-run`

**Ожидание:** План изменений без применения

**Проверка:**
- [ ] Показан план
- [ ] Вывод содержит ℹ️
- [ ] Файлы не изменены

---

## Функциональные тесты

### func-1: Обнаружение нового файла

**Подготовка:**
1. Создать новый .md файл в папке с README
2. НЕ добавлять в README

**Команда:** `/docs-reindex --check`

**Ожидание:** Обнаружен отсутствующий в индексе

**Проверка:**
- [ ] Файл указан как "Добавить"
- [ ] README помечен как требующий обновления

**Очистка:** Удалить файл

---

### func-2: Обнаружение удалённого файла

**Подготовка:**
1. В README есть ссылка на несуществующий файл

**Команда:** `/docs-reindex --check`

**Ожидание:** Обнаружена устаревшая запись

**Проверка:**
- [ ] Запись указана как "Удалить"
- [ ] README помечен как требующий обновления

---

### func-3: Интеграция с links-validate

**Команда:** `/docs-reindex --verbose`

**Ожидание:** Вызван links-validate

**Проверка:**
- [ ] В выводе есть статистика ссылок
- [ ] Битые ссылки перечислены (если есть)

---

## Edge cases

### edge-1: Пустая папка

**Подготовка:** Папка без .md файлов

**Команда:** `/docs-reindex {пустая-папка}`

**Ожидание:** Корректная обработка

**Проверка:**
- [ ] Нет ошибок
- [ ] Вывод: "Нет файлов для индексации"

---

### edge-2: Циклические ссылки в README

**Подготовка:** README A ссылается на README B, B ссылается на A

**Команда:** `/docs-reindex`

**Ожидание:** Корректная обработка без зацикливания

**Проверка:**
- [ ] Скилл завершился успешно
- [ ] Нет бесконечного цикла

---

### edge-3: Большой проект

**Команда:** `/docs-reindex` (на полном проекте)

**Ожидание:** Обработка всех файлов

**Проверка:**
- [ ] Показано общее количество файлов
- [ ] Время выполнения разумное
