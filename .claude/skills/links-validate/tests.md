# Тесты: links-validate

## Smoke-тесты

### smoke-1: Базовый запуск

**Команда:** `/links-validate`

**Ожидание:** Вывод статистики проверки ссылок

**Проверка:**
- [ ] Показано количество проверенных файлов
- [ ] Показано количество проверенных ссылок
- [ ] Показан результат (валидные/битые)

---

### smoke-2: Проверка конкретного файла

**Команда:** `/links-validate CLAUDE.md`

**Ожидание:** Проверены только ссылки в CLAUDE.md

**Проверка:**
- [ ] Файлов проверено: 1
- [ ] Вывод содержит результат

---

## Функциональные тесты

### func-1: Обнаружение битой ссылки

**Подготовка:**
1. Создать временный файл с битой ссылкой:
   ```markdown
   [несуществующий](./not-exists.md)
   ```

**Команда:** `/links-validate {temp-file}`

**Ожидание:** Битая ссылка обнаружена

**Проверка:**
- [ ] Статус: ❌ Битых: 1
- [ ] В таблице указан файл и причина

**Очистка:** Удалить временный файл

---

### func-2: Валидация относительных ссылок

**Подготовка:**
1. Файл A со ссылкой на файл B (существующий)

**Команда:** `/links-validate {файл-A}`

**Ожидание:** Ссылка валидна

**Проверка:**
- [ ] ✅ Валидных: 1
- [ ] ❌ Битых: 0

---

### func-3: Режим --dry-run

**Команда:** `/links-validate --dry-run`

**Ожидание:** Только вывод, без изменений

**Проверка:**
- [ ] Вывод содержит ℹ️
- [ ] Файлы не изменены

---

### func-4: Режим --json

**Команда:** `/links-validate --json`

**Ожидание:** JSON вывод

**Проверка:**
- [ ] Вывод — валидный JSON
- [ ] Содержит поля: files_checked, links_checked, valid, broken

---

## Edge cases

### edge-1: Пустой проект

**Подготовка:** Нет .md файлов

**Команда:** `/links-validate`

**Ожидание:** Корректная обработка

**Проверка:**
- [ ] Файлов проверено: 0
- [ ] Ссылок проверено: 0
- [ ] Нет ошибок

---

### edge-2: Внешние ссылки

**Подготовка:** Файл с внешней ссылкой `[google](https://google.com)`

**Команда:** `/links-validate {файл}`

**Ожидание:** Внешние ссылки пропущены

**Проверка:**
- [ ] Внешняя ссылка не в списке битых
- [ ] Вывод содержит информацию о пропуске

---

### edge-3: Якорные ссылки

**Подготовка:** Файл со ссылкой на несуществующий якорь `[раздел](#not-exists)`

**Команда:** `/links-validate {файл}`

**Ожидание:** Битый якорь обнаружен

**Проверка:**
- [ ] Ссылка помечена как битая
- [ ] Причина: "Якорь не найден"
