### Чек-лист проверок

- [ ] Скилл запускается по команде `/glossary-link`
- [ ] Загружается глоссарий из `/specs/glossary.md`
- [ ] Находятся markdown-файлы для обработки (исключая glossary.md)
- [ ] Первое вхождение термина в разделе заменяется на ссылку
- [ ] Формат ссылки: `[📖 Термин](/specs/glossary.md#термин)`
- [ ] Не заменяются: заголовки, блоки кода, существующие ссылки
- [ ] Выводится отчёт: количество файлов, добавленных ссылок
- [ ] Параметр `--dry-run` показывает изменения без применения
- [ ] При обработке конкретного файла запрашивается подтверждение `[Y/n]`
