# Безопасное удаление документа

Скилл для удаления документов с автоматическим созданием задачи на обновление связанных документов.

## Проблема

При удалении документа:
- Остаются битые ссылки в других документах
- Нарушается цепочка зависимостей
- Теряется контекст для связанных документов

## Решение

`/doc-delete` автоматически:
1. Находит все документы, ссылающиеся на удаляемый
2. Создаёт временный файл со списком связей
3. Создаёт задачу в `llm_tasks/current/` для актуализации

---

## Когда использовать

**Триггеры:**
- Команда `/doc-delete`
- Запрос "удалить документ" или "удалить дискуссию"
- Прямой вызов с путём к файлу

**Контекст:**
- Нужно удалить документ из `general_docs/`
- Важно не оставить битых ссылок

---

## Инструкции

### Шаг 1: Определить документ для удаления

Получить путь к файлу:
```
/doc-delete general_docs/01_discuss/001_example.md
```

Или спросить пользователя:
```
Какой документ нужно удалить?
```

### Шаг 2: Найти зависимые документы

Использовать Grep для поиска ссылок:

```bash
# Поиск по имени файла
grep -r "001_example" general_docs/ --include="*.md"

# Поиск по ID
grep -r "\[001\]" general_docs/ --include="*.md"
```

### Шаг 3: Создать временный файл со связями

Создать файл в `llm_tasks/temp/`:

```markdown
# Связи для удаляемого документа

**Удаляемый файл:** general_docs/01_discuss/001_example.md
**Дата:** YYYY-MM-DD

## Найденные ссылки

| Файл | Строка | Контекст |
|------|--------|----------|
| 02_architecture/001_*.md | 15 | [Дискуссия](../01_discuss/001_example.md) |
| 04_decisions/DEC-001_*.md | 8 | Из дискуссии 001 |

## Рекомендации

1. Обновить архитектуру — убрать ссылку или указать "удалена"
2. Обновить ADR — проверить актуальность решения
```

### Шаг 4: Спросить пользователя

```
Найдены ссылки на документ в 3 файлах:
- 02_architecture/001_ui_styleguide.md
- 04_decisions/DEC-001_ui_framework.md
- 000_discuss.md (индекс)

Варианты:
1. Удалить документ и создать задачу на обновление связей
2. Отменить удаление
3. Показать все найденные ссылки
```

### Шаг 5: Удалить документ (если подтверждено)

**ВАЖНО:** Использовать соответствующий скрипт удаления!

```bash
# Для дискуссий
python scripts/discuss_delete.py 001 --force

# Для ADR
python scripts/decision_delete.py DEC-001 --force

# Для ресурсов
python scripts/resource_delete.py 001 -t backend --force

# Для планов
python scripts/imp_plan_delete.py 001 --force
```

**ЗАПРЕЩЕНО:** Удалять файлы напрямую через rm или unlink!

### Шаг 6: Создать задачу на актуализацию

```bash
python scripts/task_new.py -t "Актуализировать связи после удаления [файл]" -c docs -p medium
```

Содержимое задачи:
```markdown
# DOCS-XXXXX: Актуализировать связи после удаления [файл]

## Описание

Удалён файл `general_docs/01_discuss/001_example.md`.
Необходимо обновить связанные документы.

## Файлы для обновления

1. `02_architecture/001_ui_styleguide.md` — убрать ссылку на дискуссию
2. `04_decisions/DEC-001_ui_framework.md` — проверить актуальность

## Связанный контекст

См. `llm_tasks/temp/deleted_001_example_links.md`

## Критерии завершения

- [ ] Все ссылки обновлены или удалены
- [ ] Нет битых ссылок (проверить make docs-health)
- [ ] Индексы актуальны
```

## Типы документов и скрипты удаления

| Тип | Папка | Скрипт |
|-----|-------|--------|
| Дискуссия | `01_discuss/` | `discuss_delete.py` |
| Архитектура | `02_architecture/` | (нужно создать) |
| ADR | `04_decisions/` | `decision_delete.py` |
| Ресурс | `05_resources/*/` | `resource_delete.py` |
| План | `06_imp_plans/` | `imp_plan_delete.py` |

## Пример использования

**Запрос:** Удалить дискуссию 001_ui_staylgayd.md

**Шаг 1:** Найти связи
```bash
grep -r "001_ui" general_docs/ --include="*.md"
```

Результат:
- `02_architecture/001_ui_styleguide.md` — ссылается
- `000_discuss.md` — в индексе

**Шаг 2:** Спросить пользователя
```
Документ 001_ui_staylgayd.md связан с:
- Архитектура: 001_ui_styleguide.md
- Индекс: 000_discuss.md

Удалить и создать задачу на обновление? (y/N)
```

**Шаг 3:** Удалить
```bash
python scripts/discuss_delete.py 001 --force
```

**Шаг 4:** Создать задачу
```bash
python scripts/task_new.py -t "Актуализировать связи: удалена дискуссия 001" -c docs -p medium
```
