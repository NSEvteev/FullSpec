- [ ] Вызывается по команде `/specs-index`
- [ ] Вызывается по фразе "обнови индексы specs"
- [ ] Вызывается по фразе "переиндексируй specs"

### Проверка поиска README

- [ ] Находит `/specs/discussions/README.md`
- [ ] Находит `/specs/impact/README.md`
- [ ] Находит `/specs/services/{service}/README.md`
- [ ] Находит `/specs/services/{service}/adr/README.md`
- [ ] Находит `/specs/services/{service}/plans/README.md`

### Проверка таблиц

- [ ] Discussions/Impact: таблица содержит #, Тема, Статус, Дата
- [ ] ADR: таблица содержит #, Тема, Impact, Статус, Дата
- [ ] Plans: таблица содержит План, ADR, Статус, Дата

### Проверка флагов

- [ ] `--dry-run` показывает изменения без применения
- [ ] `path` позволяет обновить конкретный README
