# Скрипты

Python и bash скрипты для автоматизации задач Claude Code.

## Назначение

- Утилиты для скиллов
- Вспомогательные скрипты
- Инструменты анализа

## Структура

```
/.claude/scripts/
├── README.md                    # Этот файл
├── find_references.py           # Поиск ссылок в файлах
├── update_links.py              # Массовая замена ссылок
└── {другие скрипты}
```

## Скрипты

| Скрипт | Описание | Использование |
|--------|----------|---------------|
| [find_references.py](./find_references.py) | Поиск всех ссылок на файл | `python find_references.py <путь>` |
| [update_links.py](./update_links.py) | Массовая замена ссылок | `python update_links.py <старый> <новый>` |

## Правила

1. **Naming:** `snake_case.py` или `kebab-case.sh`
2. **Документация:** Docstring в начале файла
3. **Зависимости:** Только стандартная библиотека Python
4. **Тестирование:** Тесты в `tests/` если нужны
