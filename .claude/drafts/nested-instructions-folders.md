# Драфт: Поддержка вложенных папок в README инструкций

## Проблема

**Текущая ситуация:**
- `.claude/.instructions/` содержит подпапки `skills/` и `rules/`
- README описывает только содержимое `skills/`, полностью игнорируя `rules/`
- Стандарт `standard-readme.md` (§ 3) не предусматривает описания вложенных папок

**Пример проблемы:**
```
/.claude/.instructions/
├── README.md          # Описывает только skills/, игнорирует rules/
├── skills/            # ✅ Упомянута (но неправильно — показаны файлы, а не папка)
│   └── ...
└── rules/             # ❌ Полностью проигнорирована
```

---

## Анализ

### Два типа README папок инструкций

| Тип | Содержимое | Пример |
|-----|------------|--------|
| Листовой | Только файлы инструкций | `.structure/.instructions/`, `.claude/.instructions/skills/` |
| Агрегирующий | Вложенные папки инструкций + (опционально) файлы | `.claude/.instructions/` |

### Текущая структура (standard-readme.md § 3)

5 фиксированных секций — все описывают ФАЙЛЫ:
1. Стандарты (standard-*.md)
2. Воркфлоу (create-*.md, modify-*.md)
3. Валидация (validation-*.md)
4. Скрипты (.scripts/)
5. Скиллы (ссылки на /skill-*)

**Проблема:** Нет раздела для ПАПОК (вложенных подобластей инструкций).

---

## Решение

### Структура README папок инструкций — 2 раздела

| Раздел | Описание | Когда присутствует |
|--------|----------|-------------------|
| **A. Инструкции вложенных объектов** | Описание подпапок инструкций | Если есть вложенные папки |
| **B. Инструкции объекта** | Файлы текущей папки (секции 1-5) | Всегда |

### Раздел A: Инструкции вложенных объектов

**Условие:** Присутствует если в папке `.instructions/` есть подпапки (кроме `.scripts/`).

**Формат:**

```markdown
---

## Вложенные области

Инструкции разделены на подобласти:

| Область | Описание | Индекс |
|---------|----------|--------|
| [skills/](./skills/) | Инструкции для скиллов | [README](./skills/README.md) |
| [rules/](./rules/) | Инструкции для rules | [README](./rules/README.md) |

---
```

**Правила:**
- Секция располагается ПЕРЕД секциями 1-5
- Таблица содержит все подпапки (кроме `.scripts/`)
- Ссылка ведёт на README подпапки

### Раздел B: Инструкции объекта (секции 1-5)

Стандартные 5 секций — описывают файлы ТЕКУЩЕЙ папки:
1. Стандарты
2. Воркфлоу
3. Валидация
4. Скрипты
5. Скиллы

**Правило:** Если все файлы инструкций находятся в подпапках — секции 1-5 содержат `*Нет ... — см. вложенные области.*`

### 6. validate-structure.py

**Добавить проверку:**

```python
def check_nested_areas(instructions_path: Path) -> list[str]:
    """Проверить секцию 'Вложенные области' в README инструкций."""
    errors = []
    readme_path = instructions_path / "README.md"

    # Найти подпапки (кроме .scripts)
    subfolders = [d for d in instructions_path.iterdir()
                  if d.is_dir() and d.name != ".scripts"]

    if not subfolders:
        return errors  # Нет подпапок — проверка не нужна

    # Проверить наличие секции "Вложенные области"
    content = readme_path.read_text()
    if "## Вложенные области" not in content:
        errors.append(f"S012: Секция 'Вложенные области' отсутствует в {readme_path}")
        return errors

    # Проверить что все подпапки указаны
    for subfolder in subfolders:
        if f"[{subfolder.name}/]" not in content:
            errors.append(f"S010: Подпапка {subfolder.name}/ не указана в 'Вложенные области'")

    return errors
```
