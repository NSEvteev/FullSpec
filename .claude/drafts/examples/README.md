## 1. Назначение

Папка содержит черновики, которые:
- Хорошо структурированы
- Содержат полезные паттерны
- Могут служить примерами для будущих запросов к LLM

**Как использовать:**
- При создании нового черновика — сослаться на пример из этой папки
- В промптах — показать пример желаемого формата

**Не инструкции:** Эта папка НЕ имеет зеркала в `.instructions/`. Файлы здесь — примеры, а не стандарты.

## 2. Файлы

### [example-cross-standards-ssot-analysis.md](./example-cross-standards-ssot-analysis.md)

**Анализ дублирования между стандартами и план SSOT-рефакторинга.**

Пример черновика для задачи устранения дублирования контента между документами. Демонстрирует:
- Структуру анализа (проблема → решение → порядок выполнения)
- Формат описания миграций (M1, M2, M3...)
- Чек-лист с отметками выполнения

**План валидации стандартов .github/.instructions/.**
