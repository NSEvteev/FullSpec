## Содержание

### Результаты валидации

```
python .instructions/.scripts/validate-instruction.py .github/.instructions/development/create-dev.md
```

| Код | Ошибка | Описание |
|-----|--------|----------|
| I022 | Отсутствует секция Оглавление | Нет `## Оглавление` с ссылками на секции |
| I027 | Отсутствуют нумерованные шаги | Нет `### Шаг N:` — обязательно для воркфлоу-инструкций |

### Текущая структура файла

```
---
frontmatter (ok)
---
# Воркфлоу запуска разработки
## Воркфлоу (делегация к § 0 standard-development.md)
## Чек-лист
## Скиллы
```

### Что нужно сделать

1. **Добавить `## Оглавление`** — со ссылками на все секции документа
2. **Добавить нумерованные шаги `### Шаг N:`** — либо расписать шаги из § 0 standard-development.md, либо оформить делегацию как единственный шаг
3. **Решить вопрос о дублировании** — файл сейчас делегирует к § 0, нужно ли его превратить в полноценную инструкцию или оставить как тонкий делегатор с формальным соответствием

### Вариант исправления (делегатор)

