# Черновики

Временные рабочие файлы Claude: планы, заметки, исследования.

## Назначение

- **Планы из planMode** — черновики планов реализации
- **Заметки** — временные записи в процессе работы
- **Исследования** — результаты анализа перед принятием решений

## Характеристики

| Аспект | Значение |
|--------|----------|
| В git | Да (история сохраняется) |
| Привязка | К рабочим процессам |
| Жизненный цикл | Временные (черновики) |

## Именование файлов

```
YYYY-MM-DD-<topic>.md
```

Примеры:
- `2024-01-23-auth-refactor.md`
- `2024-01-24-api-versioning.md`

## После завершения работы

Черновики можно:
1. **Оставить** — для истории решений
2. **Удалить** — если информация перенесена в финальные документы
3. **Переместить** — в `/specs/` если стал полноценной спецификацией
