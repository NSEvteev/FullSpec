## 3. Скиллы для создания

На основе воркфлоу нужно создать скиллы:

### 3.1. /structure-create

**Источник:** `workflow-create.md`

**Воркфлоу:**
1. Создать папку
2. Вызвать `generate-readme.py` → получить шаблон
3. Заполнить плейсхолдеры
4. Записать README.md
5. Обновить SSOT (/.structure/README.md)
6. Вызвать `validate-structure.py`

**Параметры:**
- `путь` — путь к новой папке (обязательный)
- `--description` — описание папки

### 3.2. /structure-modify

**Источник:** `workflow-modify.md`

**Варианты:**
- `/structure-modify rename <old> <new>` — переименование
- `/structure-modify move <path> <new-parent>` — перемещение
- `/structure-modify delete <path>` — удаление

**Воркфлоу (rename):**
1. Найти все ссылки
2. Переименовать папку
3. Обновить SSOT
4. Обновить README папки
5. Вызвать `/links-update`
6. Вызвать `validate-structure.py`

### 3.3. /structure-validate

**Источник:** `validation-structure.md`

**Воркфлоу:**
1. Вызвать `validate-structure.py`
2. Показать результат
3. (--fix) Предложить исправления

**Параметры:**
- `--fix` — предложить исправления
- `--json` — JSON вывод
