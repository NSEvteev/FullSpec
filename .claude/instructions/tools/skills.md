# Индекс скиллов

Скиллы — команды для автоматизации повторяющихся задач.

## Правила

### Правило одного действия

Скилл выполняет **1 действие** над объектом:
- `create` — создание
- `update` — обновление
- `delete` — удаление

### Формат названия

```
{объект}-{действие}
```

Примеры: `skill-create`, `agent-update`, `doc-delete`

### Расположение

```
/.claude/skills/{название}/
    SKILL.md
```
