# AI-агенты Claude Code

Эта директория предназначена для пользовательских AI-агентов Claude Code.

## Что такое агенты?

Агенты — это специализированные конфигурации Claude Code для выполнения определённых задач. В отличие от скиллов (одноразовые команды), агенты могут выполнять сложные многошаговые операции.

## Структура агента

Каждый агент находится в отдельной папке:

```
.claude/agents/
├── my-agent/
│   ├── AGENT.md          # Конфигурация и инструкции агента
│   └── context/          # Дополнительные файлы контекста (опционально)
└── another-agent/
    └── AGENT.md
```

## Пример агента

**Файл: `.claude/agents/code-reviewer/AGENT.md`**

```markdown
---
name: code-reviewer
description: Проводит код-ревью и предлагает улучшения
model: sonnet
allowed-tools: Read, Grep, Glob
---

# Агент для код-ревью

## Задача

Провести детальное ревью кода и предложить улучшения.

## Инструкции

1. Найти все файлы с кодом в проекте
2. Проанализировать каждый файл на:
   - Качество кода и читаемость
   - Потенциальные баги
   - Проблемы безопасности
   - Соответствие best practices
   - Производительность
3. Создать отчет с найденными проблемами
4. Предложить конкретные улучшения

## Критерии оценки

- Соблюдение принципов SOLID
- Обработка ошибок
- Наличие комментариев
- Тестовое покрытие
- Оптимизация производительности
```

## Использование агентов

Агенты вызываются через CLI Claude Code:

```bash
# Запустить агента
claude agent code-reviewer

# Запустить с параметрами
claude agent code-reviewer --path src/components

# Список доступных агентов
claude agents list
```

## Популярные примеры агентов

### 1. Code Reviewer
Автоматическое ревью кода с рекомендациями.

### 2. Documentation Generator
Генерация документации на основе кода.

### 3. Test Generator
Создание unit-тестов для существующего кода.

### 4. Refactoring Assistant
Помощь в рефакторинге с сохранением функциональности.

### 5. Security Auditor
Поиск уязвимостей безопасности в коде.

### 6. Performance Analyzer
Анализ производительности и предложения по оптимизации.

## Отличие от скиллов

| Характеристика | Скиллы | Агенты |
|----------------|--------|--------|
| Сложность | Простые задачи | Сложные многошаговые задачи |
| Длительность | Быстрое выполнение | Может занять время |
| Интерактивность | Минимальная | Может запрашивать уточнения |
| Контекст | Ограниченный | Полный доступ к проекту |
| Использование | `claude skill name` | `claude agent name` |

## Создание своего агента

1. Создайте папку в `.claude/agents/`:
   ```bash
   mkdir -p .claude/agents/my-agent
   ```

2. Создайте файл `AGENT.md` с конфигурацией

3. Используйте агента:
   ```bash
   claude agent my-agent
   ```
