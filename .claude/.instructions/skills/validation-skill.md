- При рефакторинге
- В CI/CD pipeline

## Шаги

### Шаг 1: Запустить валидацию

**Один скилл:**
```bash
python .claude/.instructions/skills/.scripts/validate-skill.py {skill-name}
```

**Все скиллы:**
```bash
python .claude/.instructions/skills/.scripts/validate-skill.py --all
```

**JSON вывод:**
```bash
python .claude/.instructions/skills/.scripts/validate-skill.py --json {skill-name}
```

**Пример вывода:**
```
✅ structure-create — валидация пройдена (48 строк)

❌ old-skill — 3 ошибки:
   K012: Отсутствует SSOT-ссылка
   K030: Дублирование шагов из инструкции (строки 25-89)
   K031: Скилл 156 строк (макс 80)
```

### Шаг 2: Исправить ошибки

Использовать таблицу [Типичные ошибки](#типичные-ошибки) для исправления.

## Чек-лист

### Frontmatter
- [ ] `name` присутствует, kebab-case
- [ ] `description` заполнено
- [ ] `allowed-tools` указаны
- [ ] `triggers.commands` минимум 1
- [ ] `triggers.phrases` ru и en

### Структура
- [ ] Один заголовок H1
- [ ] SSOT-ссылка сразу после заголовка
- [ ] Секция "Формат вызова" с таблицей
- [ ] Секция "Воркфлоу" с "→ Выполнить из SSOT"
- [ ] Секция "Чек-лист" со ссылкой
- [ ] Секция "Примеры" минимум 1

### Ссылки
- [ ] SSOT-файл существует
- [ ] Якорь #чек-лист валиден

### Антипаттерны
- [ ] Нет дублирования шагов
- [ ] Размер < 80 строк

## Типичные ошибки

### Frontmatter (K001-K006)

| Код | Описание | Решение |
|-----|----------|---------|
| K001 | Отсутствует `name` | Добавить поле `name` |
| K002 | Неверный формат `name` | Использовать kebab-case |
| K003 | Отсутствует `description` | Добавить описание |
| K004 | Отсутствует `allowed-tools` | Указать инструменты |
| K005 | Отсутствует `triggers.commands` | Добавить команду |
| K006 | Отсутствует `triggers.phrases` | Добавить фразы ru/en |

### Структура (K010-K016)

| Код | Описание | Решение |
|-----|----------|---------|
| K010 | Отсутствует заголовок H1 | Добавить `# {Название}` |
| K011 | Несколько заголовков H1 | Оставить один |
| K012 | Отсутствует SSOT-ссылка | Добавить `**SSOT:**` |
| K013 | Отсутствует "Формат вызова" | Добавить секцию |
| K014 | Отсутствует "Воркфлоу" | Добавить секцию |
| K015 | Отсутствует "Чек-лист" | Добавить секцию |
| K016 | Отсутствует "Примеры" | Добавить примеры |

### Ссылки (K020-K021)

| Код | Описание | Решение |
|-----|----------|---------|
| K020 | SSOT-файл не существует | Исправить путь |
| K021 | Битая ссылка в чек-листе | Исправить якорь |

### Антипаттерны (K030-K031)

| Код | Описание | Решение |
|-----|----------|---------|
| K030 | Дублирование шагов из инструкции | Заменить на "→ Выполнить из SSOT" |
| K031 | Скилл > 80 строк | Сократить, вынести в SSOT |
