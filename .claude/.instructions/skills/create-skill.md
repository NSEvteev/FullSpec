## Принципы

> **Скилл = триггер + ссылка на SSOT-инструкцию.** Скилл НЕ содержит логику — он только определяет команду и указывает на инструкцию.

> **Сначала инструкция, потом скилл.** Нельзя создать скилл без SSOT-инструкции.

**Создавать скилл когда:**

| Критерий | Условие |
|----------|---------|
| SSOT-инструкция | ✅ Существует |
| Воркфлоу | ✅ Есть в инструкции |
| Повторяемость | > 2 раз |

**НЕ создавать скилл когда:**

- ❌ Нет SSOT-инструкции → сначала создать инструкцию
- ❌ Разовое действие
- ❌ Простое действие без воркфлоу

## Шаги

### Шаг 1: Проверить SSOT-инструкцию

```
Вопросы:
1. Инструкция существует?
2. Есть секция "Шаги"?
3. Есть секция "Чек-лист"?
```

Если любой пункт не выполнен → вызвать `/instruction-create`.

### Шаг 2: Определить имя и триггеры

**Имя скилла:** формат `{object}-{action}`

| Объект | Действие | Имя скилла |
|--------|----------|------------|
| structure | create | `structure-create` |
| links | validate | `links-validate` |
| skill | modify | `skill-modify` |

**Триггеры:**
- Команда: `/{skill-name}`
- Фразы ru: 2-3 варианта
- Фразы en: 1-2 варианта

### Шаг 3: Создать папку и файл

```bash
mkdir -p .claude/skills/{skill-name}
```

Создать `SKILL.md` по шаблону из [standard-skill.md § 4](./standard-skill.md#4-секции-документа).

### Шаг 4: Заполнить SKILL.md

**Frontmatter:** SSOT → [standard-frontmatter.md § 2](/.structure/.instructions/standard-frontmatter.md#2-дополнительные-поля-для-скиллов)

**Секции:**

1. `**SSOT:**` — ссылка на инструкцию
2. `## Формат вызова` — синтаксис + таблица параметров
3. `## Воркфлоу` — предупреждение + "→ Выполнить из SSOT"
4. `## Чек-лист` — ссылка на SSOT#чек-лист
5. `## Примеры` — 2-3 примера вызова

### Шаг 5: Валидация

```bash
python .claude/.instructions/skills/.scripts/validate-skill.py {skill-name}
```

### Шаг 6: Обновить README

Добавить скилл в таблицу `/.claude/skills/README.md`.

## Чек-лист

### Подготовка
- [ ] SSOT-инструкция существует
- [ ] В инструкции есть шаги
- [ ] В инструкции есть чек-лист
- [ ] Имя скилла уникально

### Создание
- [ ] Создана папка `/.claude/skills/{name}/`
- [ ] Создан `SKILL.md` по шаблону
- [ ] Заполнен frontmatter
- [ ] Добавлена SSOT-ссылка
- [ ] Заполнен "Формат вызова"
- [ ] Заполнен "Воркфлоу"
- [ ] Заполнен "Чек-лист"
- [ ] Добавлены примеры

### Проверка
- [ ] Валидация пройдена
- [ ] README обновлён

## Примеры

### Создание скилла structure-create

```bash
# 1. Проверить SSOT-инструкцию
cat .structure/.instructions/create-structure.md

# 2. Создать папку скилла
mkdir -p .claude/skills/structure-create

# 3. Создать SKILL.md по шаблону

# 4. Валидация
python .claude/.instructions/skills/.scripts/validate-skill.py structure-create

# 5. Обновить README
```
