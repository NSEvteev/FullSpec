# Правила README папок

Форматы README для разных типов папок.

---

## Типы README

| Тип | Файл правил | Где используется |
|-----|-------------|------------------|
| [Папка проекта](./project-folder.md) | `project-folder.md` | `/src/`, `/platform/`, etc. |
| [Папка инструкций](./instructions-folder.md) | `instructions-folder.md` | `/.claude/.instructions/*` |

---

## Ключевое различие

| Аспект | README проекта | README инструкций |
|--------|----------------|-------------------|
| **Главная секция** | Зона ответственности | Индекс правил |
| **Содержит** | ЧТО хранится | КАК работать |
| **Формат** | IN + Границы | Таблица + секции |
