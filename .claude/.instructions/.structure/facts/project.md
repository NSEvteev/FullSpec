# Формат project.md

Правила формата и обновления файла `/.structure/project.md`.

---

## Назначение

Файл `/.structure/project.md` содержит полное дерево папок проекта с комментариями.

**SSOT:** [/.structure/project.md](/.structure/project.md)

---

## Формат

```markdown
# Структура папок проекта

---

## Корневые файлы

\`\`\`
/
├── README.md                    # Описание
├── CLAUDE.md                    # Описание
└── ...
\`\`\`

---

## Дерево папок (полное)

\`\`\`
/
├── src/                         # Комментарий
│   └── {service}/
│       ├── backend/
│       │   └── *.ts             #   Комментарий с отступом
...
\`\`\`
```

---

## Правила

1. **Дерево в code block** — используй ``` для дерева
2. **Комментарии справа** — `# комментарий` после элемента
3. **Отступы для вложенных** — `#   комментарий` (3 пробела) для вложенных
4. **Паттерны файлов** — `*.ts`, `*.md` вместо конкретных имён
5. **{placeholders}** — `{service}`, `{name}` для переменных частей

---

## Когда обновлять

| Событие | Действие |
|---------|----------|
| Создана папка | Добавить в дерево |
| Удалена папка | Убрать из дерева |
| Переименована папка | Обновить путь |

---

## Пример добавления

**До:**
```
├── src/
│   └── auth/
```

**После:**
```
├── src/
│   ├── auth/
│   └── notify/                  # Новый сервис уведомлений
```
