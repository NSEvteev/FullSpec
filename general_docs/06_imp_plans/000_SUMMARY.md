## Правила обновления

При создании или изменении плана реализации:

1. **Определить область** плана
2. **Добавить/обновить строку** в таблицу "Планы реализации"
3. **Добавить/обновить ссылку** в соответствующую секцию "Планы по областям"
4. **Обновить счётчики** в таблице "Быстрый обзор по областям"
5. **Добавить/обновить блок** в секцию "Краткое описание планов"

**Скилл `/summary`** автоматически выполняет эти шаги.

**Workflow:**
```
ADR (approved) → /imp-plan → План (draft) → ... → План (final) → /folder-doc
```
