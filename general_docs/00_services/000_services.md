## Статусы сервисов

| Статус | Описание |
|--------|----------|
| `draft` | Сервис идентифицирован, описание создаётся |
| `in_progress` | Идёт разработка документации и реализации |
| `active` | Сервис работает в production |
| `deprecated` | Сервис устарел, планируется замена |
