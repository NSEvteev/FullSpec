## Назначение

Детальное описание всех ресурсов системы.

**[📖 Ресурс](../glossary.md#ресурс)** — компонент системы (БД, сервис, API), который:
- Описан детально (схема, API, конфигурация)
- Определен в архитектуре
- Используется в реализации
