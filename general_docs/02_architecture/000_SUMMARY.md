## Правила обновления

При переходе архитектуры в статус `🟢 approved`:

1. **Определить область** компонента (UI, Backend, Auth, Database, Infra, Architecture)
2. **Добавить строку** в таблицу "Архитектурные решения" с указанием области
3. **Добавить ссылку** в соответствующую секцию "Решения по областям"
4. **Обновить счётчик** в таблице "Быстрый обзор по областям"
5. **Добавить блок** в секцию "Краткое описание решений"
