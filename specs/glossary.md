## Термины

### Система инструкций

| Термин | Определение |
|--------|-------------|
| SSOT | Single Source of Truth — единственный авторитетный источник правды для определённого типа информации |
| Артефакт | Файл с определённой структурой, подчиняющийся стандарту |
| Workflows | Документы, напрямую зависящие от стандарта (create-, modify-, validation-) |
| Экземпляры | Файлы, созданные по стандарту |
| Frontmatter | YAML-метаданные в начале markdown-файла между `---` |
| standard-version | Версия стандарта в frontmatter документа |
| Миграция | Процесс обновления зависимых файлов при изменении стандарта (9 шагов) |
| Скилл | Slash-команда для выполнения типовой операции (например, `/instruction-create`) |
| Rule | Контекстное правило, автоматически загружаемое при работе с файлами |
| Агент | Специализированный помощник для автономных задач через Task tool |
| Черновик | Временный документ для исследований и планирования (frontmatter опционален) |
