# Стратегия тестирования

Кросс-сервисная стратегия — типы тестов, мокирование, межсервисные тесты.

## 1. Типы тестов

| Тип | Когда | Где |
|-----|-------|-----|
| Unit | Логика одного модуля | `src/{svc}/tests/unit/` |
| Integration | Взаимодействие с БД/кешем | `src/{svc}/tests/integration/` |
| E2E | Сквозные сценарии | `tests/e2e/` |
| Smoke | Базовая работоспособность | `tests/smoke/` |
| Load | Производительность | `tests/load/` |
