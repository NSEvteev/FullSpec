## 1. Когда валидировать

**Обязательно:**
- Перед коммитом нового скрипта
- После существенных изменений скрипта
- При code review

**Опционально:**
- В CI/CD pipeline
- При рефакторинге

## 2. Что проверяется

### Структура файла

| Проверка | Обязательно | Описание |
|----------|:-----------:|----------|
| Shebang | ✅ | `#!/usr/bin/env python3` в первой строке |
| Docstring | ✅ | Многострочный docstring после shebang |
| Расширение | ✅ | Файл `.py` |
| Кодировка | ✅ | UTF-8 |

### Docstring

| Проверка | Обязательно | Описание |
|----------|:-----------:|----------|
| Первая строка | ✅ | `{name}.py — {описание}.` |
| Секция "Использование" | ✅ | Присутствует |
| Секция "Примеры" | ✅ | Минимум 1 пример |
| Секция "Возвращает" | ✅ | Exit codes описаны |

### Код

| Проверка | Обязательно | Описание |
|----------|:-----------:|----------|
| `if __name__` | ✅ | Точка входа |
| `main()` функция | ✅ | Основная функция |
| UTF-8 Windows | ⚠️ | Рекомендуется |
| `sys.exit()` | ⚠️ | Рекомендуется |

## 3. Коды ошибок

| Код | Категория | Описание |
|-----|-----------|----------|
| **S001** | Shebang | Отсутствует shebang |
| **S002** | Shebang | Неверный формат shebang |
| **S010** | Docstring | Отсутствует docstring |
| **S011** | Docstring | Отсутствует первая строка описания |
| **S012** | Docstring | Неверный формат первой строки |
| **S013** | Docstring | Отсутствует секция "Использование" |
| **S014** | Docstring | Отсутствует секция "Примеры" |
| **S015** | Docstring | Отсутствует секция "Возвращает" |
| **S020** | Структура | Отсутствует `if __name__` |
| **S021** | Структура | Отсутствует функция `main()` |
| **S030** | Стиль | Отсутствует UTF-8 настройка для Windows |
| **S031** | Стиль | Отсутствует `sys.exit()` |

## 4. Запуск валидации

### Ручная проверка

```bash
python .instructions/.scripts/validate-script.py path/to/script.py
```

### Проверка всех скриптов

```bash
python .instructions/.scripts/validate-script.py --all
```

### Вывод

```
✅ script.py — валидация пройдена
❌ broken.py — 2 ошибки:
   S010: Отсутствует docstring
   S020: Отсутствует `if __name__`
```

## 5. Чек-лист

```markdown
## Чек-лист валидации скрипта

- [ ] Shebang `#!/usr/bin/env python3`
- [ ] Docstring с первой строкой `{name}.py — {описание}.`
- [ ] Секция "Использование" в docstring
- [ ] Секция "Примеры" в docstring (минимум 1)
- [ ] Секция "Возвращает" в docstring
- [ ] Функция `main()`
- [ ] `if __name__ == "__main__": main()`
- [ ] UTF-8 для Windows в `main()`
- [ ] `sys.exit()` с кодом возврата
```
