## Шаги

### Шаг 1: Проверить структуру файла

**Автоматически:**
```bash
python .instructions/.scripts/validate-script.py {path}
```

**Вручную (если нужно):**

| Проверка | Обязательно | Описание |
|----------|:-----------:|----------|
| Shebang | ✅ | `#!/usr/bin/env python3` в первой строке |
| Docstring | ✅ | Многострочный docstring после shebang |
| Расширение | ✅ | Файл `.py` |
| Кодировка | ✅ | UTF-8 |

### Шаг 2: Проверить docstring

| Проверка | Обязательно | Описание |
|----------|:-----------:|----------|
| Первая строка | ✅ | `{name}.py — {описание}.` |
| Секция "Использование" | ✅ | Присутствует |
| Секция "Примеры" | ✅ | Минимум 1 пример |
| Секция "Возвращает" | ✅ | Exit codes описаны |

### Шаг 3: Проверить код

| Проверка | Обязательно | Описание |
|----------|:-----------:|----------|
| `if __name__` | ✅ | Точка входа |
| `main()` функция | ✅ | Основная функция |
| UTF-8 Windows | ⚠️ | Рекомендуется |
| `sys.exit()` | ⚠️ | Рекомендуется |

### Шаг 4: Проверить принципы

> **Проверки на соответствие принципам** из [standard-principles.md](./standard-principles.md).

**Валидация:** [validation-principles.md](./validation-principles.md)

| Принцип | Проверка | Код ошибки |
|---------|----------|------------|
| **KISS** | Нет переусложнённых конструкций | P001 |
| **DRY** | Нет дублирования кода | P002 |
| **YAGNI** | Нет неиспользуемых аргументов/функций | P003 |
| **Минимум зависимостей** | Нет тяжёлых библиотек (pandas, requests) | P004 |
| **Обработка ошибок** | Нет голого `except:` | P005 |
| **Читаемость** | Docstring для функций | P006 |

**При обнаружении нарушений:**

1. Зафиксировать нарушения
2. Исправить скрипт
3. Вызвать `/instruction-modify` для связанных инструкций (если интерфейс изменился)

### Шаг 5: Проверить отображение в инструкции

| Проверка | Обязательно | Описание |
|----------|:-----------:|----------|
| Есть в секции "Скрипты" | ✅ | Скрипт указан в связанной инструкции |
| Ссылка валидна | ✅ | Путь к файлу корректен |
| Назначение актуально | ✅ | Соответствует первой строке docstring |

## Чек-лист

### Структура
- [ ] Shebang `#!/usr/bin/env python3`
- [ ] Docstring с первой строкой `{name}.py — {описание}.`
- [ ] Секция "Использование" в docstring
- [ ] Секция "Примеры" в docstring (минимум 1)
- [ ] Секция "Возвращает" в docstring
- [ ] Функция `main()`
- [ ] `if __name__ == "__main__": main()`

### Код
- [ ] UTF-8 для Windows в `main()`
- [ ] `sys.exit()` с кодом возврата
- [ ] Нет голого `except:`

### Принципы
- [ ] KISS — простой код
- [ ] DRY — нет дублирования
- [ ] YAGNI — нет лишнего функционала
- [ ] Минимум зависимостей — только stdlib + PyYAML
- [ ] Docstring для каждой функции

### Интеграция
- [ ] Скрипт добавлен в секцию "Скрипты" инструкции
- [ ] Назначение в таблице соответствует docstring

## Типичные ошибки

### Структура (S0xx)

| Ошибка | Код | Причина | Решение |
|--------|-----|---------|---------|
| Нет shebang | S001 | Отсутствует shebang | Добавить `#!/usr/bin/env python3` |
| Неверный shebang | S002 | Неверный формат | Исправить на `#!/usr/bin/env python3` |
| Нет docstring | S010 | Отсутствует docstring | Добавить docstring |
| Нет описания | S011 | Отсутствует первая строка | Добавить `{name}.py — {описание}.` |
| Неверный формат описания | S012 | Неверная первая строка | Формат: `{name}.py — {описание}.` |
| Нет "Использование" | S013 | Отсутствует секция | Добавить секцию |
| Нет "Примеры" | S014 | Отсутствует секция | Добавить секцию |
| Нет "Возвращает" | S015 | Отсутствует секция | Добавить секцию |
| Нет `if __name__` | S020 | Отсутствует точка входа | Добавить блок |
| Нет `main()` | S021 | Отсутствует функция | Добавить функцию |
| Нет UTF-8 | S030 | Нет настройки Windows | Добавить reconfigure |
| Нет `sys.exit()` | S031 | Нет кода возврата | Добавить `sys.exit()` |

### Принципы (P0xx)

| Ошибка | Код | Причина | Решение |
|--------|-----|---------|---------|
| Переусложнённый код | P001 | Нарушение KISS | Упростить конструкции |
| Дублирование кода | P002 | Нарушение DRY | Вынести в функцию |
| Неиспользуемый код | P003 | Нарушение YAGNI | Удалить лишнее |
| Тяжёлая зависимость | P004 | pandas, requests и т.д. | Заменить на stdlib |
| Голый except | P005 | `except:` без типа | Указать тип исключения |
| Нет docstring функции | P006 | Функция без документации | Добавить docstring |

## Скрипты

| Скрипт | Назначение | Инструкция |
|--------|------------|------------|
| [validate-script.py](./.scripts/validate-script.py) | Валидация формата скрипта | Этот документ |

**Использование:**
```bash
# Проверка одного скрипта
python .instructions/.scripts/validate-script.py path/to/script.py

# Проверка всех скриптов
python .instructions/.scripts/validate-script.py --all

# JSON вывод
python .instructions/.scripts/validate-script.py --json path/to/script.py

# Включая проверку принципов
python .instructions/.scripts/validate-script.py --principles path/to/script.py
```

**Вывод:**
```
✅ script.py — валидация пройдена
❌ broken.py — 3 ошибки:
   S010: Отсутствует docstring
   S020: Отсутствует `if __name__`
   P005: Голый `except:` в строке 45
```
