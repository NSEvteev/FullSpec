## Обзор

**Скилл:** [/instruction-deactivate](/.claude/skills/instruction-deactivate/SKILL.md)

**Принцип:** Инструкции НЕ удаляются. Они деактивируются.

**Всего шагов:** 10

---

## Когда использовать

- Инструкция больше не актуальна для проекта
- Технология/практика больше не используется
- Инструкция заменена другой

**Пример:** [shared/i18n.md](../shared/i18n.md) — i18n не используется в проекте.

---

## Шаги

### Шаг 1: Получить путь

Получить путь к инструкции из параметров вызова.

**Формат:** `{папка}/{название}.md` относительно `/.claude/.instructions/`

---

### Шаг 2: Проверить существование

```bash
# Проверка существования файла
if [ ! -f "/.claude/.instructions/{путь}" ]; then
    echo "ОШИБКА: Инструкция не найдена"
    exit 1
fi
```

**При отсутствии:** остановить выполнение.

---

### Шаг 3: Прочитать содержимое

```bash
Read → /.claude/.instructions/{путь}
```

**Извлечь:**
- Frontmatter (сохраняется)
- Основное содержимое (будет закомментировано)

---

### Шаг 4: Добавить предупреждение

После frontmatter добавить блок:

```markdown
> **ВАЖНО:** Эта инструкция не используется в проекте.
>
> Содержимое закомментировано. Для реактивации — раскомментировать.
```

---

### Шаг 5: Закомментировать содержимое

Обернуть всё содержимое (после frontmatter и предупреждения) в HTML-комментарий:

```markdown
<!--
# Оригинальное содержимое

## Секция 1
...

## Секция 2
...
-->
```

**Важно:** Frontmatter остаётся незакомментированным.

---

### Шаг 6: Сохранить файл

```bash
Write → /.claude/.instructions/{путь}
```

Файл сохраняется с:
- Оригинальным frontmatter
- Предупреждением
- Закомментированным содержимым

---

### Шаг 7: Обновить README папки

**Путь:** `/.claude/.instructions/{папка}/README.md`

**Что обновить:**
- Отметить инструкцию как неактивную
- Добавить пометку `(не используется)`

**Пример:**
```markdown
| Файл | Статус |
|------|--------|
| [i18n.md](./i18n.md) | ⚠️ Не используется |
```

---

### Шаг 8: Сбросить статус в главном README

**Путь:** [/.claude/.instructions/README.md](/.claude/.instructions/README.md)

**Что обновить:**
- Убрать инструкцию из активных
- Добавить в секцию неактивных (если есть)
- Или пометить статус

---

### Шаг 9: Проверка по чек-листу

Пройти по чек-листу (см. ниже). Все пункты должны быть выполнены.

---

### Шаг 10: Результат

```
✅ Инструкция деактивирована

Файл: /.claude/.instructions/{путь}
Статус: Не используется

README папки обновлён
Главный README обновлён
```

## Формат деактивированной инструкции

```markdown
---

  - {связанные инструкции}
---

> **ВАЖНО:** Эта инструкция не используется в проекте.
>
> Содержимое закомментировано. Для реактивации — раскомментировать.

<!--
# Оригинальный заголовок

{Оригинальное содержимое}

## Секция 1
...

## Секция 2
...
-->
```
