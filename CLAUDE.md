# CLAUDE.md

Справочная информация о проекте для Claude Code.

---

## Структура

```
/.claude/
├── skills/         # Скиллы (команды /...)
├── rules/          # Правила автозагрузки контекста
├── .instructions/  # Как писать скиллы и rules
├── agents/         # Агенты
└── drafts/         # Черновики

/.instructions/     # Как писать инструкции
```

---

## Команды

```bash
make dev    # Запустить для разработки
make test   # Запустить тесты
make lint   # Линтинг
```

---

## Правила

Контекст загружается автоматически из `/.claude/rules/` при работе с файлами.

При предложении вариантов выбора использовать `AskUserQuestion` tool.
